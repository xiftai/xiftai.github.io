POLITIQUE DE COOKIES

La web que está visitando utiliza cookies para mejorar la navegación y obtener datos estadísticos.

Las cookies son archivos de texto que se instalan en el equipo desde el que se accede a esta página y que envían, de forma automática, información a nuestro servidor. Las cookies no son virus informáticos. No dañan su ordenador ni ralentizan su funcionamiento. Puede eliminarlas en cualquier momento, o rechazarlas configurando el programa de navegación que utiliza.

Si no desea recibir estas cookies, puede configurar su navegador para que sean rechazadas. En caso contrario, entendemos que contamos con su consentimiento para su instalación.

Para conocer cómo administrar cookies en el navegador, por favor visite los enlaces siguientes:

  • Firefox desde http://support.mozilla.org/es/kb/habilitar-y-deshabilitar-cookies-que-los-sitios-we
  • Chrome desde http://support.google.com/chrome/bin/answer.py?hl=es&answer=95647
  • Explorer desde http://windows.microsoft.com/es-es/windows7/how-to-manage-cookies-in-internet-explorer-9
  • Safari desde http://support.apple.com/kb/ph5042

Si rechaza su instalación o las borra, no podrás disfrutar de una experiencia completa en la navegación de este sitio web.

Google Analytics

Este sitio web utiliza Google Analytics, el servicio de análisis web. Este servicio lo presta Google Inc., 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA.

Google Analytics utiliza “cookies” Las cookies son ficheros de texto almacenados en tu dispositivo que permiten realizar un estudio sobre cómo utilizas este sitio web. La información generada sobre el uso de este sitio web y almacenada en la cookie se trasmite a los servidores de Google en USA y se almacena allí.

IP’s anónimas

Hemos activado las IP anónimas en este sitio web. Tu dirección IP será acortada por Google dentro de la Unión Europa así como en otros lugares que hayan firmado el Acuerdo con el Espacio Económico Europeo antes de trasmitir la información a Estados Unidos. Sólo en casos excepcionales se enviará la dirección IP completa al servidor de Google en USA y será acortada allí.

Google usará esa información para evaluar el uso de este sitio web, para realizar informes sobre la actividad del sitio web y para para proveer otros servicios relacionados con la actividad del sitio web y el uso de Internet. La IP que se trasmite por tu navegador como parte de Google Analytics no se utilizará para hacer un cruce de datos con otros datos que Google almacene.
